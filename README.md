# Smart Helmet for Enhanced Rider Safety

A final year B.Sc. IT project | Thakur College of Science & Commerce, Mumbai

## About
An IoT-based smart helmet that integrates sensors and wireless 
communication to prevent accidents and send emergency alerts automatically.

## Features
- Alcohol detection using MQ-3 sensor — blocks ignition if drunk
- Helmet wear verification before bike starts
- Accident detection using MPU6050 accelerometer & gyroscope
- Real-time GPS tracking using NEO-6M module
- Automatic SMS emergency alert via SIM800L GSM module
- SOS button for manual emergency trigger
- LED indicators and buzzer alerts

## Components Used
- ESP32 Microcontroller
- MQ-3 Alcohol Sensor
- MPU6050 Accelerometer & Gyroscope
- NEO-6M GPS Module
- SIM800L GSM Module
- Pressure/Proximity Sensor
- Li-ion Rechargeable Battery

## Technologies
IoT | Embedded Systems | C/C++ | ESP32 | GSM Communication
